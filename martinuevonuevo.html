EdgeAI · Marine Recovery · HackUPC 2026

From Blind Spots
to Smart Reefs

Barcelona's 14 km of breakwaters — the escolleres — are potential sanctuaries for marine biodiversity. Today, nobody knows which sections are thriving and which are dying. BlauEdge fixes that.

We deploy EdgeAI sensor nodes directly on the breakwater rocks. Each node runs a FOMO object-detection model on an Arduino UNO Q to count 5 native and invasive fish species in real time — measuring temperature and pH simultaneously. No cloud. No connectivity required. Continuous 24/7.

🌡️
The Problem
Thermal heatwaves push temps past 28 °C — bleaching Posidonia seagrass and wiping out juvenile fish in hours, not days. Invasive Siganus luridus displace native Diplodus sargus with no one tracking the shift. Manual surveys cost €10K+ and happen quarterly at best.
🧠
The Hardware
Each node uses a Modulino Thermo (water temp via I²C), a Modulino pH (acidification events), and a USB camera running Edge Impulse FOMO at <200 ms latency. INT8 quantized. <100 KB flash. No custom PCBs — deployable today.
📡
The Solution
Python backend receives Serial JSON → calculates biodiversity score (0–100) → runs root-cause diagnosis → serves live data to this dashboard via Flask API. Every restoration decision is now data-driven and surgical, multiplying impact and slashing waste.
