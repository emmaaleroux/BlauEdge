The Problem

Barcelona's coast is changing — and we're blind to it

The escolleres can either devastate or become unlikely sanctuaries for marine biodiversity. But without continuous monitoring, every restoration decision is a guess.

🌡️
Thermal stress events
Mediterranean heatwaves push water temperatures past 28°C — triggering bleaching events fatal for Posidonia seagrass and juvenile fish populations sheltering in the rocks. These spikes last hours, not days. Nobody is logging them.
🔇
Acoustic silence
Healthy breakwaters buzz with biological activity — crabs, fish, mussels, urchins. That vibration signature is detectable by sensors. When it goes quiet, the ecosystem has collapsed. Today, nobody is listening.
💸
Wasted restoration
Marine biologists spend €10K+ per survey, done quarterly at best. Repopulation of native species is scattered across zones with no data — many efforts fail because the zone isn't ready to support life.
The BlauEdge insight

If we identify the best recovery zones in real-time, we can direct repopulation of Posidonia oceanica, sea urchins, and bivalves with surgical precision — multiplying impact and slashing waste. Better biodiversity means healthier fish stocks, more fishing income, and CO₂ sequestration through blue carbon.

Hardware

The sensor node — built from the hackathon kit

Each BlauEdge node uses standard Arduino Modulino components in a waterproof enclosure bolted to the breakwater surface. No custom PCBs. Deployable today.

🧠
Arduino UNO Q
The brain. Runs Edge Impulse INT8 quantized 1D Conv model in <200ms. No cloud. No connectivity required. Processes all sensor data locally.
🌡️
Modulino Thermo
Water temperature proxy. Detects thermal stress events above 28°C — the threshold at which Posidonia bleaching begins. Critical for early-warning.
💡
Modulino Light
Water clarity and light penetration depth proxy. Low lux signals turbidity spikes or algae blooms — both indicators of ecosystem stress.
〰️
Modulino Vibro
Acoustic proxy — the most novel sensor. Marine life creates distinct low-frequency vibrations: fish, crustaceans, bivalves. Biological silence = ecosystem collapse.
📷
USB Web Camera
Surface colonization detection via FOMO object detection. Measures visible algae coverage, mussel colonies, barnacle density — a second AI layer.
🔔
Pixels + Buzzer
RGB LED displays health state color in real-time. Buzzer triggers on STRESSED classification. Field researchers know status without a laptop.
Sensor → Ecosystem mapping: Temperature → thermal stress · Light → turbidity/bloom · Vibration → bioacoustics · Camera → surface colonization
