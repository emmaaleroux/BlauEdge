The Problem

Barcelona's coast is changing — and we're blind to it

The escolleres can either devastate or become unlikely sanctuaries for marine biodiversity. But without continuous monitoring, every restoration decision is a guess.

🌡️
Thermal stress events
Mediterranean heatwaves push water temperatures past 28°C — triggering bleaching events fatal for Posidonia seagrass and juvenile fish populations sheltering in the rocks. These spikes last hours, not days. Nobody is logging them.
🐟
Invisible fish decline
Native species like Diplodus sargus and Scorpaena scrofa are vanishing from urban reefs — displaced by invasive Lessepsian migrants like Siganus luridus. Without continuous AI monitoring, nobody tracks this shift in real time.
💸
Wasted restoration
Marine biologists spend €10K+ per survey, done quarterly at best. Repopulation of native species is scattered across zones with no data — many efforts fail because the zone isn't ready to support life.
The BlauEdge insight

If we identify the best recovery zones in real-time, we can direct repopulation of Posidonia oceanica, sea urchins, and bivalves with surgical precision — multiplying impact and slashing waste. Better biodiversity means healthier fish stocks, more fishing income, and CO₂ sequestration through blue carbon.

Hardware

The sensor node — built from the hackathon kit

Each BlauEdge node uses an Arduino UNO Q with Modulino sensors and a USB camera, in a waterproof enclosure bolted to the breakwater surface. No custom PCBs. Deployable today.

🧠
Arduino UNO Q
The brain. Runs Edge Impulse FOMO object detection model for fish species counting. All inference on-device. No cloud. No connectivity required.
🌡️
Modulino Thermo
Water temperature via I²C. Detects thermal stress events — triggers diagnosis engine when temp rises significantly above historical baseline.
📷
USB Web Camera
FOMO object detection counts Diplodus sargus, Scorpaena scrofa, Oblada melanura, Siganus luridus, and Aurelia aurita per frame in real time.
🧪
Modulino pH
Analog pH sensor. Acidification below 8.1 — the Mediterranean optimum — triggers the root-cause diagnosis engine and flags Besòs/runoff events.
🐠
Edge Impulse Model
FOMO-based object detection trained on 5 breakwater species. INT8 quantized. Outputs bounding boxes + labels → Python counts detections per species.
💻
Python Backend
Receives Serial JSON, stores to CSV, calculates biodiversity score (0–100), runs root-cause diagnosis, and serves live data via Flask API on :5050.
Data flow: Camera → Edge Impulse FOMO → species counts → Serial JSON → Python → CSV → Flask API → this dashboard

EdgeAI Model

On-device inference — no cloud, ever

The model runs entirely on the Arduino UNO Q. A FOMO (Faster Objects, More Objects) object detection model trained in Edge Impulse Studio, quantized to INT8, detects and counts 5 marine species from each camera frame.
