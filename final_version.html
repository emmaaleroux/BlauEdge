From Blind Spots
to Smart Reefs

Barcelona's 14 km of breakwaters — the escolleres — are potential sanctuaries for marine biodiversity. Today, nobody knows which sections are thriving and which are dying. BlauEdge fixes that.

We deploy EdgeAI sensor nodes directly on the breakwater rocks. Each node runs a FOMO object detection model on an Arduino UNO Q to count native and invasive fish species in real time — measuring temperature and pH simultaneously. No cloud. No connectivity required. Continuous 24/7.

🌡️
The Problem
Extreme heatwaves are pushing ocean temperatures to lethal limits, triggering mass die-offs and ecological collapse in a matter of hours. As invasive species aggressively displace native wildlife, our current monitoring systems—crippled by high costs and slow data—are failing to track the devastation. We are losing entire marine ecosystems before we even realize they are gone.
🧠
The Hardware
Each node uses a Modulino Thermo to get temperature of the water, a Modulino pH to account for acidification events, and a USB camera running Edge Impulse FOMO.
📡
The Solution
Raw Serial Data → AI-Driven Biodiversity Scoring (0–100) → Instant Root-Cause Diagnosis → Live Flask API Dashboard We’ve turned chaotic data into surgical action. By replacing guesswork with real-time intelligence, every restoration effort is now precision-engineered to multiply impact and obliterate waste.
